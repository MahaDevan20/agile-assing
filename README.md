# SDG Assignment - EduConnect

This project contains the assignment submission for the Agile Development & CASE Tool Lab assignment.

## Files Created

1. **SDG_Assignment_Submission.html** - Assignment documentation (ready to convert to PDF)
2. **prototype.html** - HTML prototype of the EduConnect platform
3. **convert_to_pdf.py** - Script to convert HTML to PDF (requires additional setup on Windows)

## Converting HTML to PDF

### Method 1: Using Browser (Recommended - Easiest)

1. Open `SDG_Assignment_Submission.html` in any web browser (Chrome, Edge, Firefox)
2. Press `Ctrl + P` (or go to File → Print)
3. Select "Save as PDF" or "Microsoft Print to PDF" as the destination
4. Click "Save"
5. Choose a location and save as `SDG_Assignment_Submission.pdf`

### Method 2: Using Python Script

The `convert_to_pdf.py` script uses weasyprint which requires additional system libraries on Windows. For easier conversion, use Method 1 above.
